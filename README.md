# Express DB App

A express app with SQLite db.

## Application structure
- `public/` - App static files
- `src/` - App backend
- `views/` - App frontend

## Application features
This app offers:
- Route protection
- API protection
- User register
- User login
- User log out
- Database storage

## How to use

### Preparing the environment

To run the application, you must have Node.js version 24 LTS installed.
- [Node 24LTS](https://nodejs.org/pt-br/download)

### Install and run

Now you need to install the dependencies.

```bash
# On the project directory
npm install

```

Finally, you can run the project.

```bash
# Run in normal mode
npm run start

# Run in watch mode
npm run start:watch
```
